# HeyEmma.ai Landing Page - PRD

## Original Problem Statement
Create a landing page for HeyEmma.ai using the uploaded logo and content structure from https://supabook.ai/ai-answering-service/

## User Choices
- Static marketing/landing page only
- Content adapted for HeyEmma as AI answering service
- Dark theme (matching logo's black background)
- ElevenLabs Convai widget in hero section

## Target Audience
Business owners who miss calls and need AI answering service

## Core Requirements (Static)
- Modern, professional dark-themed landing page
- Logo integration (cyan/purple color scheme)
- ElevenLabs voice widget for live demo
- Multiple sections showcasing product value

## What's Been Implemented (Feb 5, 2025)

### Components Built
1. **Navbar** - Logo, navigation links (Features, How it Works, Voice Cloning, FAQ), CTA buttons
2. **Hero** - Split layout with headline + ElevenLabs widget
3. **Problem Section** - 4 pain point cards (missed calls, voicemails, late replies, unanswered questions)
4. **Features Section** - "More than an answering service" with 5 feature items
5. **How it Works** - 4-step flow + 3 benefit cards
6. **Voice Cloning** - Voice sample visualization with play button
7. **Bento Features** - 4-card grid (AI Phone, Follow-ups, CRM, 24/7)
8. **Comparison** - 8-feature comparison table
9. **FAQ** - 8-question accordion
10. **Final CTA** - 4 guarantee cards + main CTA
11. **Footer** - Logo, links, social icons

### Tech Stack
- React 19 + Tailwind CSS
- Framer Motion for animations
- Shadcn/UI components (Accordion, Button)
- ElevenLabs Convai Widget integration

### Design System
- Fonts: Outfit (headings), Inter (body)
- Colors: Cyan (#06b6d4), Purple (#a855f7), Pink (#ec4899)
- Background: #030303
- Glass-morphism cards with backdrop blur
